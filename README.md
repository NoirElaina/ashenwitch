# NoirElaina Blog

个人博客源码仓库。

## Description

这是 `NoirElaina` 的个人博客项目，基于 `Astro` 构建，用来记录开发学习、项目实践和日常折腾。

在线地址：

- `https://noirelaina.github.io`

## Stack

- `Astro`
- `Svelte`
- `Tailwind CSS`
- `Pagefind`

## Local Development

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm check
pnpm build
```

## Deployment

项目通过 GitHub Actions 自动部署到 GitHub Pages。

推送到 `main` 分支后会自动触发部署，发布地址为：

- `https://noirelaina.github.io`
